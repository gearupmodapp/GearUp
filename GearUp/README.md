# GearUp Modding Toolkit

License: GPLv3

An open-source modding toolkit designed for asset extraction, modification, and rebuilding. GearUp provides the necessary tools to unpack game files, convert them into editable formats, and repack them back into the game.

---

## How It Works

1. Extraction: The toolkit extracts the core game files from the main directory.
2. Conversion: Extracted assets are converted into standard formats, making them ready for editing, texture modification, or mesh adjustment.
3. Repacking: After editing, the assets are converted back into their original game format.
4. Build: The toolkit builds the modified files directly into the game structure.

---

## Installation and Usage

To install and run the toolkit:
1. Click the green "**Code**" button on this repository page and select "**Download ZIP**".
2. Copy and paste all the contents of the archive directly into the main installation folder of the game.
3. Run GearUp.exe to launch the application.

---

## Requirements and Recommended Software

* Operating System: Windows 10 / 11
* 3D Modeling (Recommended): Blender (for editing and modifying the converted asset meshes)
* Texture Editing (Recommended): GIMP (for modifying and editing image textures)
